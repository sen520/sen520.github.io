###### 读书

- 算法

  - 《漫画算法：小灰的算法之旅》

  - 《算法图解》

    大多是理论性的知识，值得一看，也提供了 很多其他优秀的算法，但需要查找资料自行了解

  - 《大话数据结构》未读完

- web

  - 《JavaScript语言精粹》
  - 《Django企业开发实战》

- 服务

  - 《Docker技术入门与实践》

- 数据

  - 《SQL基础教程》
  - 《MySQL必知必会》* 很不错的一本书
  - 《MongoDB权威指南》
  - 《elasticsearch权威指南》

- R

  - 《R语言实战》未读完

- python

  - 《利用python进行数据分析》

  - 《python网络数据采集》

  - 《疯狂的python讲义》

  - 《python数据可视化》

  - 《流畅的python》未读完

    偏底层，只阅读部分高级函数和类的用法

- 爬虫

  - 《精通Scrapy网络爬虫》
  - 《python3网络爬虫开发实战》

- other

  - 《成为极少数》
